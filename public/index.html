🛒 Price Compare API

Welcome to the Price Compare API - your intelligent product price comparison service for Pakistani shoppers.

📡 API Endpoint

POST /api/compare

📝 Request Format

{
  "product": "iPhone 15 Pro"
}

✨ Features

🌐 Multi-Source

Searches across Daraz, OLX, Goto, Symbios, iShopping, Amazon, AliExpress, and more

🔍 AI-Powered

Uses Claude AI with web search to find real-time prices

🛡️ Trust Scores

Provides trust scores and manipulation risk assessment for each listing

💰 Price Analysis

Calculates lowest price, average price, and provides buying recommendations

🚀 Example Usage

curl -X POST https://your-domain.vercel.app/api/compare \
  -H "Content-Type: application/json" \
  -d '{"product": "Samsung Galaxy S24"}'

📦 Response Format

{
  "product": "Product name",
  "lowestPrice": "XXX PKR",
  "averagePrice": "XXX PKR",
  "sites": [
    {
      "name": "Website Name",
      "url": "https://...",
      "price": "price string",
      "priceNum": 1234,
      "trustScore": 85,
      "manipulationRisk": "Low",
      "riskReason": "brief reason"
    }
  ],
  "advice": "Buying recommendation"
}

ℹ️ About

This API is part of the Price Compare PK mobile application, helping Pakistani shoppers make informed purchasing decisions by comparing prices across multiple e-commerce platforms.
